# Roadmap

A practical plan for the web-design agency (Alex + Brady). The homesite and a
couple of templates exist; this is how we get from "site on disk" to "paying clients."

Status legend: ✅ done · 🟡 in progress · ⬜ not started

---

## Phase 1 — Brand & Identity  🟡
The one real blocker. Everything public-facing waits on a name.

- ⬜ **Pick the agency name.** ("Groundwork" is out — domain taken.) Check `.com`
  availability *before* committing (Namecheap / Cloudflare).
- ⬜ Replace the `[Agency Name]` placeholder everywhere (homesite + all templates).
- ⬜ Lock a **color palette** + confirm fonts (currently Fraunces + Inter, minimal B/W).
- ⬜ Make a simple **logo / wordmark**; replace `assets/favicon.svg`.
- ⬜ Decide a **business email** (e.g. hello@yourname.com) and repoint the contact form.

## Phase 2 — Finish the Homesite  🟡
- ✅ Full structure: hero, stats, services, templates band, pricing, work, about, FAQ, contact
- ✅ Animations (scroll-reveal, hero entrance, count-up), hover polish, reduced-motion support
- ⬜ Real **project screenshots** in `assets/` (mineralcalc, portfolio, lawncare, pools)
- ⬜ Add a real **og-image.png** (1200×630) for link previews
- ⬜ Tighten **About** copy with real bios + a photo of each of you
- ⬜ Proofread all copy; confirm pricing tiers match what you'll actually charge

## Phase 3 — Template Library  🟡
Each template = a sales asset and a faster build for that industry.

- ✅ Restaurant (Italian / family)
- ✅ Home services (HVAC / plumbing)
- ⬜ **Lawncare / landscaping** — Brady's niche; highest-priority next build
- ⬜ **Pools** — you already have a real client; template doubles as the deliverable
- ⬜ Salon / barbershop / spa (booking-focused)
- ⬜ Auto detailing / repair
- ⬜ Capture **screenshot thumbnails** of each template for the gallery cards
- ⬜ (Optional) a shared starter stylesheet so new templates spin up faster

## Phase 4 — Go Live  ⬜
- ⬜ Buy the **domain** for the agency name
- ⬜ Deploy the static site — **Netlify, Cloudflare Pages, or GitHub Pages** (all free)
- ⬜ Wire the contact form to a real inbox — **Formspree** or **Web3Forms** (see README)
- ⬜ Add **Google Analytics / Plausible** to see traffic
- ⬜ Basic **SEO**: titles, meta descriptions, sitemap.xml, submit to Google Search Console
- ⬜ Test on real phones (iOS + Android) and a couple of browsers

## Phase 5 — Get Clients  ⬜
The part that actually makes money.

- ⬜ Define the **offer + packages** in writing (what each tier includes, timeline, revisions)
- ⬜ Build a simple **intake checklist** (what you need from a client to start)
- ⬜ Set up a **payment method** (Stripe, PayPal, or invoices) — consider 50% upfront
- ⬜ Make a **target list**: local businesses with bad/no sites (drive around, Google Maps,
  Facebook). Lawncare + pools first since Brady already has reach there.
- ⬜ **Outreach**: short personalized email/DM + a mockup or a link to the matching template
- ⬜ Ask first happy clients for a **testimonial** + permission to feature their site
- ⬜ Lean on Brady's **lawncare app** user base as a warm channel

## Phase 6 — Operate & Grow  ⬜
- ⬜ Standardize the **build process** (kickoff → draft → revisions → launch → handoff)
- ⬜ Offer an optional **maintenance plan** ($X/mo for updates) for recurring revenue
- ⬜ Track jobs in a simple sheet (lead → quote → deposit → live → paid)
- ⬜ Reinvest: more templates, better portfolio shots, maybe paid local ads once profitable
